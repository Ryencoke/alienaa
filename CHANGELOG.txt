FIX: GAME AJAX "NETWORK ERROR" + UNIFIED BUY BUTTON COLOR
-----------------------------------------------------------
[index.php]
- CRITICAL FIX: mining, vats, and the Exchange vault kept hitting "Network
  error" on every action. Root cause (present since the BATCH-27 mining
  rewrite): their JSON endpoints live inside the page files, which index.php
  only requires AFTER printing the entire page layout — so every AJAX
  response was "<!DOCTYPE html>...{json}" and JSON parsing failed.
  New game-AJAX gateway: POSTs carrying mine_ajax / vat_ajax / exch_ajax are
  routed to the page file BEFORE any HTML output (with a jail check that
  returns a clean JSON error for suspended accounts). Mining digs/moves,
  vat planting/harvesting, and vault cracks now return pure JSON.

  accent teal (rgba accent bg/border + accent text) — instead of per-page

  rarity-colored), Stock Exchange "Buy" (was green), Black Market "⚡ Bid"
