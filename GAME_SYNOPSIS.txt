WHAT IS SPRAWL-9?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Sprawl-9 is a browser-based cyberpunk MMO text RPG. Players create a
character and live in a dystopian city called "The Sprawl" — doing jobs,
fighting, trading, managing resources, and climbing the ranks. The game
is multi-player and persistent.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
TECH STACK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- Flat PHP (NO framework, NO MVC — intentional)
- MySQL database
- Entry point: index.php (AJAX nav — swaps main.center, dispatches sprawl:swapped)
- Pages: /pages/*.php (included by index.php via ?p=pagename)
- Helpers: lib.php (chat_color, nav_links, role_label, etc.)
- Config: config.php (gitignored)
- Styles: style.css (cyberpunk dark theme — neon #19f0c7, magenta #ff2d95)
- No build pipeline — plain HTML/CSS/JS inline

THEME & AESTHETIC

Cyberpunk noir. Dark backgrounds, neon accent colors, terminal-style UI.

IMPORTANT PATTERNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- attachCounters() in index.php auto-adds char counters to inputs.
  Suppress with: data-no-counter on the input/textarea.
- char counter removal: add data-no-counter to inputs that shouldn't show one.
- CSRF guard: csrf_guard() called at top of index.php for POST requests.
- Admin log: all player edits logged to admin_log via $log() closure in admin.php.
- Escrow pattern (trade.php): credits/shards/items deducted from proposer on propose,
  returned on cancel/reject, transferred on accept — all in a DB transaction.
- state_api.php: polled by JS every 4s; returns online list, chat messages, notifications.
- Daily reset: triggered manually via admin panel (manager only) or runs via cron;
  restores Health/Signal, adds +250 Drive up to cap (500 non-sub / 1500 sub).
